# Z-City: Improved Gang Wars (IGWARS)

Author: DJfarent
Base Gamemode: Z-City
Status: WORK IN PROGRESS (NOT FINISHED)

---

ABOUT

Improved Gang Wars (IGWARS) is a custom gamemode built for Z-City that expands on the standard gang war experience.

The goal is to create a more dynamic, replayable, and intense round-based mode with:

* Better weapon variety
* Expanding/contracting combat zones
* Improved spawn logic
* More balanced and engaging gameplay loops

This project is currently incomplete and under active development.

---

CURRENT FEATURES

* Custom gamemode integration with Z-City mode system
* Randomized spawn selection
* Custom HUD elements
* Basic round flow (start/end handling)
* Initial weapon loadout system
* Support for attachments, armor, and ammo configs

---

KNOWN ISSUES

* Mode selection menu may break or not display IGWARS correctly
* Zone size scaling may feel incorrect or too small
* Zone damage may not always apply properly
* Some hooks may not fire due to Z-City mode loader conflicts
* Weapon distribution lacks balancing/testing
* Spawning inconsistencies in certain maps

---

INSTALLATION

1. Place the gamemode files inside:
   zcity/gamemode/modes/

2. Ensure files follow naming convention:

   * cl_improve_gang_wars.lua
   * sh_improve_gang_wars.lua
   * sv_improve_gang_wars.lua

3. Restart server or reload Lua.
